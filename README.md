# Cobalt Mod
(Currently WIP) A mindustry mod that aims to add more unique turrets, blocks and units

# Content

- Currently adds 17 diferent turrets, those include
	
	Shrapnel turrets (Fuse-like);
	Burst turrets ;
	Air-Only turrets;
	Power turrets;
	Artillery turrets;
